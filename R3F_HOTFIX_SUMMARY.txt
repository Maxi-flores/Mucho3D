   - New component that runs INSIDE <Canvas>

   - Updates Zustand with camera position every frame
   - Returns null (invisible component for side effects only)

   - REMOVED: useThree() hook import

   - ADDED: Read cameraPosition from Zustand store
   - Now pure UI component (no R3F hooks)

   - ADDED: cameraPosition: [number, number, number]

================================================================================
                              DATA FLOW
================================================================================

CameraTracker (inside Canvas)
    ↓
    useThree() → camera object
    useFrame() → runs every render frame
    setCameraPosition() → updates Zustand
    ↓
sceneStore (Zustand)
    ↓
    stores cameraPosition: [x, y, z]
    ↓
FloatingHUD (outside Canvas)
    ↓
    reads cameraPosition from Zustand
    displays X, Y, Z values
    renders FPS and scene stats
