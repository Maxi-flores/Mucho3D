================================================================================
                         AI PIPELINE ARCHITECTURE
================================================================================

Phase 4 (Planned): Ollama + Orchestrator

User Flow:
  1. User opens GenerationChat in Studio
  2. Types prompt: "Create a modern chair"
  3. Chat shows: "Planning..."
  4. orchestrator.service calls ollama.service
  5. Ollama returns: { objects: [...], operations: [...] }
  6. Schema validation with Zod
  7. Chat shows: "Executing..." + plan preview
  8. Blender executes plan (deterministic operations)
  9. Result returned to chat
  10. "Export" or "Revise" options shown
  11. Result saved to project.generations in Firestore

Key Components:
  - ollamaService      Prompt → Structured plan
  - orchestratorService Orchestration + validation
  - GenerationChat     UI for generation interface
  - Zod Schema         Plan validation
  - Blender Worker     Deterministic execution
